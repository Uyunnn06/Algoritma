Fungsi BUBBLE_SORT(Array)
    Tentukan n = panjang Array
    UNTUK setiap elemen i dari 0 sampai n-1:
        UNTUK setiap elemen j dari 0 sampai n-i-2:
            Jika Array[j] > Array[j+1], maka:
                Tukar Array[j] dengan Array[j+1]
        AKHIR UNTUK
